# When creating or updating a skill follow below instructions

- use `$skill-creator`
- When asked to create or update a skill, make the changes in `~/.codex/skills/`, not in the `codex-skills` repository.
- When creating or updating a skill, maintain its `AGENTS.md` reference entry, following the entries for skills listed in `README.md`.
- Keep the `Custom Skills Repository` list in `README.md` up to date when skills are created or updated.
